# Employee-Histogram
🔹 Project Overview
This Power BI project analyzes employee performance ratings across different departments using histogram visualization. The dashboard helps identify rating distribution patterns, compare departmental performance, and support HR decision-making processes.

Objective

Visualize the distribution of employee ratings (1–5 scale)

Compare performance trends department-wise

Identify high-performing and low-performing departments

Support data-driven HR decisions (appraisals, promotions, training)

The dataset includes the following fields:

Employee ID

Employee Name

Department

Job Role

Experience

Performance Rating (1–5 scale)

⚙️ Tools & Technologies Used

Power BI Desktop

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)
